Modernized Data Center Network with Cisco 9000

Offering Formats

This course is offered in the following formats:

  • Live, Instructor-Led Training (schedule)

Course Overview

In this instructor-led course, you will be introduced to the Modernized Data Center Architecture. This architecture is rapidly becoming the network foundation for ISPs, Enterprise and Commercial Data Centers. This class will showcase the Cisco Nexus 9000 product line and how these products can be used to implement the Modern Data Center in addition to perform in a classic 3-Tier Architecture.

Through the use of lecture, discussion and hands-on labs, you will come to understand how the Cisco Nexus 9000 can be utilized in any Data Center Design as we go from the classic Core-Aggregation-Access (3-Tier) topology to the Modern Spine-Leaf topology running VxLAN/eVPN using MP-BGP as the control plane. Finally, using Cisco Nexus 9000 product line you will come to understand the implementation of Ciscoâ€™s Application Centric Infrastructure (ACI) for a fully Software defined and controlled Data Center Fabric. Programmability concepts will be included in the labs.

It is designed for customers, systems engineers, network engineers, and field engineers. This course includes hands-on labs with physical Nexus 9000 hardware, L2/L3 switches, and simulators as required to provide each learner with their own lab environment.

Who Should Participate?

  • Data Center Architects
  • Network Administrators / System Engineers
  • Solution Architects / System Integrators
  • End-Users Deploying & Implementing Nexus Switches

Recommended Prerequisites

  • Fundamental understanding of L2 and L3 networking
