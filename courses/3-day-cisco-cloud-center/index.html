3-Day Cisco Cloud Center

Offering Formats

This course is offered in the following formats:

  • Live, Instructor-Led Training (schedule)

Course Overview

In this 3-day Instructor-led Cisco CloudCenter Hand-On Training, you will become experienced with the planning, designing, and implementation of Cisco CloudCenter solution.

This course is designed for Cloud Architects, Sysadmins, Network Architects, Software Architects, and DevOps Engineers who will be implementing and managing Cisco’s CloudCenter. Cisco CloudCenter provides the ability to model, manage and orchestrate deployments of applications seamlessly with complete agnostic behavior to virtually any public cloud such as AWS, as well as to any private clouds such as vSphere. This course will provide hands-on experience with deploying Cisco CloudCenter with integration and orchestrated provisioning to private and public clouds.

Upon completion of the course, students will be familiar with:

  • Cisco CloudCenter Architecture
  • Cisco CloudCenter Workflows, Administration Objects & Terminology
  • Cisco CloudCenter Deployment Environment, Usage and Fees
  • Cisco CloudCenter User Administration, Sub-Tenant Configuration
  • Cisco CloudCenter Modeling and Deployment
  • Cisco CloudCenter Benchmark, Scheduling
  • Cisco CloudCenter CICD Project Board
  • Cisco CloudCenter Policies, Tagging and Governance

Course Specifics

  • This course has a high degree of labs (40:60 ratio lab:lecture), and is intended to engage and challenge the student into performing the most common but also complex scenarios when deploying Cisco CloudCenter
  • Hands on lab environments. Each student will have:
    • Dedicated CloudCenter Manager and multiple CloudCenter Orchestrators. 2 ESXi hosts and multiple VM’s
    • Dedicated vCenter Appliance
    • Dedicated ACI Tenant
  • Class size will be restricted to 16 students to facilitate more hands on lab time.
  • Each student will have a dedicated CloudCenter and ACI Tenant, dedicated ports both on the ACI Leafs and on the external Nexus 5K/6Ks, dedicated 2 ESXi servers, vCenter server and VMs.
  • Students do not share or “partner-up” to perform tasks. Each student independently performs all tasks.
